## General advices

- Understand the problem.
- Keep It Stupid Simple (KISS)
- Make it work, make it right, make it fast.
- Begin at the beginning.
- Start with tests.
- Do One Thing (DOT).
- Start small and iterate.
- Pure function > Function > Factory > Class
